# キープオン HP制作サービスサイト — ワイヤーフレーム設計書

## 本書について

本書は「キープオン」のHP制作サービスサイトのワイヤーフレームを作成するための設計書です。サイトの構造、各ページの構成要素、インタラクション仕様、デザイントーンをまとめています。

---

## 1. サイト概要

### 何のサイトか

子供向けサービス（幼稚園・保育園・習い事教室・団体・チーム等）に特化したホームページ制作サービスの事業サイト。

### 想定ユーザー

- 幼稚園・保育園の園長先生、事務担当者
- 習い事教室・学童・チームの運営者
- 年齢層：30〜60代、女性比率高め
- 閲覧環境：スマホが主（業務中はPCを開けない。休憩時間や帰宅後にスマホで情報収集する）

### サイトの目的

1. 訪問者に「自分に合ったコース」を理解してもらう（診断チャート）
2. LINEまたはフォームで問い合わせてもらう（コンバージョン）

## 5. 診断チャート詳細仕様

### 5-1. 全体構造

- トップページ内のセクション、またはモーダル（オーバーレイ）で実装
- 1問ずつ表示。選択すると次の質問にアニメーション遷移
- プログレスバーを上部に表示（1/5 → 2/5 → ... → 5/5）
- 全5問＋結果画面の計6画面

### 5-2. 各質問の仕様

#### Q1

```
┌─────────────────────────────────────────────────┐
│  ■■■□□  1/5                                      │
│                                                   │
│  今、ホームページはお持ちですか？                  │
│                                                   │
│  ┌─────────────────────────────────┐              │
│  │  ない（これから初めてつくる）     │              │
│  └─────────────────────────────────┘              │
│  ┌─────────────────────────────────┐              │
│  │  あるが、古い・スマホ未対応      │              │
│  └─────────────────────────────────┘              │
│  ┌─────────────────────────────────┐              │
│  │  あるが、もっと魅力的にしたい    │              │
│  └─────────────────────────────────┘              │
│                                                   │
└─────────────────────────────────────────────────┘
```

- 選択肢は大きなボタン（カード型）。タップしやすいサイズ（高さ最低56px）
- 選択すると0.3秒程度のフェードで次の質問に遷移
- 「戻る」リンクをプログレスバー横 or 下部に設置

#### Q2

```
質問文：ホームページを一番見てほしいのは誰ですか？

選択肢：
  - 入園・入会を検討している保護者
  - 働きたい・関わりたいと思っている求職者
  - 両方
```

#### Q3

```
質問文：ホームページで一番伝えたいことは何ですか？

選択肢：
  - 雰囲気や温かさ（写真やイラストで）
  - 教育方針やカリキュラムの内容
  - アクセスや料金などの実用的な情報
  - まだ整理できていない（一緒に考えてほしい）
```

#### Q4

```
質問文：ご予算のイメージに近いのはどちらですか？

選択肢：
  - 月額費用だけで始めたい（初期費用なし）
  - しっかり投資して、納得のいくものをつくりたい
```

#### Q5

```
質問文：ホームページ以外にあったらいいなと思うものは？

選択肢（複数選択可）：
  - チラシ・パンフレット
  - 紹介動画・ムービー
  - SNS（Instagram等）の運用サポート
  - 特になし（まずはホームページだけ）
```

- Q5のみ**複数選択可**。選択状態のトグル＋「この内容で結果を見る」ボタンで結果画面へ遷移
- 「特になし」選択時は他の選択肢を非活性にする

### 5-4. 振り分けロジック

```
テンプレートコース推奨の条件：
  Q1 = A（HPなし）AND Q4 = A（月額だけ）

オーダーメイドコース推奨の条件：
  Q4 = B（しっかり投資）
  OR Q1 = B or C（既存HP有り）AND Q3 = A or D
  OR Q3 = D（一緒に考えてほしい）

採用ページオプション追加推奨：
  Q2 = B or C（求職者を含む）

おすすめオプション表示：
  Q5の選択に応じて該当オプションを表示
  Q5 = D（特になし）の場合はオプションセクション非表示
```

```
ボタン（プライマリ）：
  背景：ブランドカラー / 文字：白 / 角丸：8px / padding：16px 32px
  ホバー：やや暗く / クリック：やや沈む

ボタン（セカンダリ）：
  背景：白 / 枠線：ブランドカラー / 文字：ブランドカラー / 角丸：8px

カード：
  背景：白 / 角丸：12px / シャドウ：0 2px 8px rgba(0,0,0,0.08)
  ホバー：シャドウ拡大（0 4px 16px rgba(0,0,0,0.12)）

アコーディオン：
  閉じた状態：質問文＋右端に「＋」アイコン
  開いた状態：質問文（太字）＋回答文展開＋アイコンが「−」に変化
  アニメーション：0.3秒のスライドダウン
```

### 7-4. アイコン・イラスト方針

- **アイコン**：線画スタイル（Line icons）、太さ2px、ブランドカラー
- **イラスト**：温かみのあるフラットイラスト。子ども・先生・保護者がモチーフ
- **写真**：明るく温かい自然光。子どもの笑顔、先生の働く姿、園の外観等
- 既存メインビジュアル（デバイスモックアップ＋パステル背景）のトーンを全体に統一

## 9. インタラクション・アニメーション

### 使用するアニメーション

| 対象 | アニメーション | 備考 |
|------|--------------|------|
| セクション表示 | フェードイン＋やや上方向からスライド | スクロール連動、控えめに |
| 診断チャート 質問遷移 | フェードアウト→フェードイン | 0.3秒 |
| 診断チャート プログレスバー | 幅が伸びるアニメーション | 0.3秒 |

| カードホバー | シャドウ拡大＋やや上に移動 | 0.2秒 |

### 使用しないもの

- パララックス（重くなりがち、SPで不具合が起きやすい）
- 自動再生動画（ページ表示速度に影響）
- 過度なスクロールアニメーション（ユーザーの集中を妨げる）

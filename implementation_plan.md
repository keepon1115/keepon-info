# キープオンHP制作事業 — サイト実装計画書 v1.0

## 本書の位置づけ

本書は、キープオン株式会社が新たに立ち上げる「子供向けサービス専門HP制作事業」の事業サイトを制作するための実装計画書である。戦略的意思決定、サービス設計、サイト構造、コピー方針、技術仕様をすべて網羅し、このドキュメント単体でサイト制作に着手できることを目的とする。

---

## 1. 事業概要

### 1-1. 事業の定義

幼稚園・保育園・習い事教室・団体・チームなど「子供向けサービス」に特化したHP制作事業。テンプレートコース（制作費無料）とオーダーメイドコースの二本立てで、非対面ファーストのプロセス設計により、現場で忙しい先生・運営者のハードルを徹底的に下げる。

### 1-2. ブランド名

「キープオン」をそのまま継続使用。既存のネットショップ事業・STEAMスクール事業との連携を活かしつつ、HP制作事業として独立したサイトを構える。

### 1-3. 事業の根幹思想

> 園のホームページは、その園の「顔」である。AIが急速に発展する時代において、ホームページがなければ園の魅力は社会に伝わりきらない。にもかかわらず、多くの園や子供向けサービスにとってHP制作は「価格が高い」「打ち合わせに時間がかかる」「何を伝えればいいかわからない」という三重のハードルがある。キープオンは、その三つのハードルを徹底的に下げることで、すべての子供向けサービスが「ちゃんとした顔」を持てる状態を届ける。

### 1-4. 最大の差別化ポイント

**「寄り添いをデジタルで再現する仕組み」**

「お客様一人一人に寄り添う」という姿勢を、サイト体験そのもので実現する。訪問者がサイトに来た瞬間から、対面で相談しているかのように「あなたにはこれが合いますよ」と導く。その象徴が、サイトトップに設置する「診断チャート」（後述）。

競合サイトは料金表を並べて「お問い合わせください」で終わっているところがほとんど。キープオンはサイト自体が寄り添いの体験になる。

### 1-5. ターゲット

- 幼稚園・保育園（認可・認可外・こども園含む）
- 習い事教室（ピアノ、スイミング、体操、英語、プログラミング、そろばん等）
- 子供向け団体・チーム（少年野球、サッカークラブ、ボーイスカウト等）
- 放課後等デイサービス、学童保育
- 子育て支援施設

共通項：保護者が「わが子を預けたい・通わせたい」と感じるかどうかが集客の生命線であり、ホームページがその判断の入口になっている。

### 1-6. 前回戦略との関係

前回の戦略では、出前ワークショップを認知装置の中心に据え、HP制作はエコシステムの一部品と位置づけていた。今回、HP制作を独立事業として切り出すことで以下の構造に変わる。

- **出前ワークショップ** → 園との最初のリアル接点（認知装置）。ここは変わらない
- **HP制作事業（本サイト）** → ワークショップで出会った園にHP制作を提案する導線＋独立した集客チャネル
- **ショップ（カラーミー）** → 継続取引による接触頻度の確保。ここも変わらない

三者の関係は「どこからでも入れて、どこにでもつながる」循環構造を維持。HP制作だけが独立サイトを持つことで、子供向けサービス業界での専門性と信頼感を強く打ち出す。

## 2. サービス設計

### 2-1. コース構成

#### コースA：テンプレートコース

| 項目 | 内容 |
|------|------|
| コンセプト | テンプレートを選び、テキストと画像を送るだけでHPが完成する最軽量コース |
| ターゲット | HPを持ったことがない園・教室、とにかくまず形にしたい運営者 |
| 初期費用 | 無料（0円） |
| 月額費用 | 5,500円〜（税込）※プランにより異なる |
| 納期目安 | 最短1週間 |
| テンプレート数 | 3〜5パターン（トーン別：やわらかナチュラル系 / 元気ポップ系 / 落ち着き信頼系） |
| 含まれるもの | テンプレートデザイン、レスポンシブ対応（PC・スマホ・タブレット）、基本ページ構成、問い合わせフォーム、LINE公式アカウント連携 |

#### コースB：オーダーメイドコース

| 項目 | 内容 |
|------|------|
| コンセプト | 園・教室の理念・雰囲気・こだわりを丁寧にヒアリングし、オリジナルデザインで制作 |
| ターゲット | リニューアルしたい園、採用にも力を入れたい園、世界観にこだわりたい教室 |
| 初期費用 | 348,000円〜（税別）※業界主力ゾーン30〜45万円を1割強下回る水準 |
| 月額保守 | 14,800円（税別）月1回更新込み |
| 納期目安 | 4〜8週間 |
| 含まれるもの | オリジナルデザイン、レスポンシブ対応、写真撮影ディレクション、採用ページ同梱可、ブログ/お知らせ機能 |

#### コース間の導線

テンプレートコースで始めた園が、後からオーダーメイドへ移行できる「ステップアップ導線」を設計する。二つのコースは上下関係ではなく、園の状況に応じた「別の選択肢」として並列に提示する。

### 2-2. オプションメニュー

| オプション | 価格目安 | 備考 |
|------------|----------|------|
| チラシ・パンフレット制作 | 30,000円〜 | 園児募集用、採用用 |
| 園紹介動画・PR動画 | 98,000円〜（HP同時契約時） | 通常150,000円〜 |
| ロゴ・オリジナルイラスト | 30,000円〜 | フリーランス連携 |
| SNS運用サポート | 月額19,800円・初期0円 | Instagram、LINE公式 |
| 月額保守・更新代行 | テンプレ：月額に含む / オーダーメイド：14,800円 | 月1回更新込み |

#### トータルパッケージ

HP＋ロゴ＋撮影＋動画＋SNS3ヶ月をセットで「各単品合計 約75万円 → パッケージ598,000円」として訴求。世界観統一と窓口一本化がベネフィット。

### 2-3. プロセス設計（非対面ファースト）

```
STEP 1：診断チャート（サイト上）
　→ 訪問者の状況を5問で把握し、最適なコースを提案

STEP 2：非対面ヒアリング
　→ ウェブフォーム or LINE で基本情報・希望イメージ・掲載内容を送付
　→ テンプレートコースの場合はテンプレート選択もここで完了

STEP 3：校正確認
　→ 初稿をオンラインで共有、テキスト・画像の修正指示を非同期でやりとり
　→ 先生が空いた時間にスマホから確認・返信できる状態

STEP 4：オンライン面談（必要に応じて）
　→ 細部の詰めや方向性確認。ここまでの文脈が共有済みのため短時間で完了

STEP 5：公開・納品
　→ 公開後の保守・更新サポート開始
```

本質：「会わなくても進められる」ことではなく、「園の先生の負担を最小化しながら、きちんと園の想いを引き出す」こと。

## 3. 診断チャート設計

### 3-1. 設計思想

サイト訪問者が最初に触れる体験であり、キープオンの「寄り添い」をデジタルで体現する仕組み。5問の質問に答えるだけで、自分に最適なコース・オプションの組み合わせが提案される。対面相談のように「あなたにはこれが合いますよ」と導く。

### 3-2. 質問設計（5問）

#### Q1. 今のホームページの状況（実務・現状把握）
**「今、ホームページはお持ちですか？」**

- A）ない（これから初めてつくる）
- B）あるが、古くなっている・スマホ対応できていない
- C）あるが、もっと魅力的にリニューアルしたい

→ 振り分けロジック：Aはテンプレート寄り、B・Cはオーダーメイド寄りに傾斜

#### Q2. 一番届けたい相手（ターゲット理解）
**「ホームページを一番見てほしいのは誰ですか？」**

- A）入園・入会を検討している保護者
- B）働きたい・関わりたいと思っている求職者
- C）両方

→ 振り分けロジック：Bを含む場合、採用ページ同梱オプションを推奨

#### Q3. ホームページで一番伝えたいこと（想いの引き出し）
**「ホームページで一番伝えたいことは何ですか？」**

- A）雰囲気や温かさ（写真やイラストで感じてもらいたい）
- B）教育方針やカリキュラムの内容
- C）アクセスや料金など、実用的な情報をわかりやすく
- D）まだ整理できていない（一緒に考えてほしい）

→ 振り分けロジック：Aはデザイン重視のオーダーメイド推奨、Cはテンプレートでも十分対応可能、Dはオーダーメイド＋ヒアリング重視を推奨

#### Q4. 予算感（実務・フィルタリング）
**「ご予算のイメージに近いのはどちらですか？」**

- A）月額費用だけで始めたい（初期費用はかけたくない）
- B）しっかり投資して、納得のいくものをつくりたい

→ 振り分けロジック：Aはテンプレートコース確定、Bはオーダーメイドコース推奨

#### Q5. ホームページ以外にほしいもの（クロスセル導線）
**「ホームページ以外に、こんなものもあったらいいなと思うものはありますか？」**

- A）チラシやパンフレット
- B）紹介動画・ムービー
- C）SNS（Instagramなど）の運用サポート
- D）特になし（まずはホームページだけ）

→ 振り分けロジック：A〜C選択時に該当オプションを推奨セットに追加

### 3-3. 結果表示パターン

診断結果は以下の形式で表示する。

```
━━━━━━━━━━━━━━━━━━━━━━
あなたにおすすめのプラン
━━━━━━━━━━━━━━━━━━━━━━

【テンプレートコース】or【オーダーメイドコース】

（選ばれた理由を2〜3行で表示）
例：「初めてのホームページづくりで、まずは月額費用だけで
始めたいとのこと。テンプレートを選ぶだけで、最短1週間で
あなたの園の顔が完成します。」

＋おすすめオプション：チラシ制作、紹介動画
（Q5の回答に基づき表示）

[ 無料で相談する → ]（LINEまたはフォームへ遷移）
━━━━━━━━━━━━━━━━━━━━━━
```

### 3-4. 実装上の注意

- 1問ずつ表示し、選択するとアニメーションで次の質問に遷移する（一覧表示ではなく対話型）
- スマホでも操作しやすいボタンサイズ（タップターゲット最低44px）
- 途中離脱を防ぐため、プログレスバー（1/5, 2/5...）を表示
- 結果画面から直接LINE問い合わせまたはフォームに遷移できる導線

    - 「まずは無料相談から。お気軽にどうぞ。」

    - ナビゲーション、会社情報、関連サイトリンク（ショップ、スクール）

## 5. コピー・メッセージ方針

### 5-1. キャッチコピー候補

メインビジュアルに載せるキャッチコピーの方向性。最終決定は制作時に選定。

- **案A（寄り添い軸）**：「あなたの園の "いいところ"、ホームページにしませんか。」
- **案B（ハードル軸）**：「テキストと写真を送るだけ。あなたの園のホームページ、つくれます。」
- **案C（想い軸）**：「わかりやすいデザインで、あなたの想いを伝えます。」（※既存サイト踏襲）
- **案D（子供向け特化軸）**：「子どもに関わるすべての場所に、ちゃんとした "顔" を。」

### 5-2. サブコピー

「幼稚園・保育園・習い事・団体・チームのホームページ制作。制作費無料のテンプレートコースから、こだわりのオーダーメイドまで。」

### 5-3. トーン＆マナー

- **温かく、親しみやすい日本語。** 専門用語を使わない。「コーディング」ではなく「ホームページづくり」、「レスポンシブ」ではなく「スマホでもきれいに見えます」
- **押し売り感を排除。** 「まずは気軽にご相談ください」「LINEで聞いてみる」のような柔らかい導線
- **保護者目線に立った説明。** 「保護者がスマホで見たとき、第一印象で "ここに通わせたい" と感じるデザインを」
- **先生・運営者への共感。** 「毎日お忙しい先生に代わって」「打ち合わせの時間が取れなくても大丈夫」
